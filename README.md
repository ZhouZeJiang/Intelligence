## 用3升,5升杯子怎么量出4升水

先将5升水倒满-然后倒入3升杯子-3升杯子把水倒掉-5升杯子再把剩余2升水倒进3升杯子-然后将5升杯子装满水-倒进3升水杯子即量完4升水

## 烧一根不均匀的绳要用一个小时，如何用它来判断一个小时十五分钟

1号绳子两头同时烧+2号绳子一头开始烧-1号绳子烧完,2号绳子烧到一半-2号绳子另一头开始烧烧完后烧3号绳子的两头一起烧烧完就是1小时15分钟

## 你只能带行走 60 公里的油，只能在起始点加油，如何穿过 80 公里的沙漠

先装60公里的油开车开到距离起点20公里放20公里的油在这-开回起点装满油然后开到20公里处将20公里油装上-就有60公里的油开完剩下60公里的沙漠

## 赛马问题，25 匹马，5 个赛道，最少几次能选出最快的三匹马

25匹马分为5组-每组比完(5次)选出第一名-选出来的五个第一名跑一次(1次)选前三-第一名那组跑的二三名再和现在的二三名跑一次(1次)-5+1+1=7次

## 五只鸡五天能下五个蛋，一百天下一百个蛋需要多少只鸡

五只鸡一天一个蛋-所以一百天一百个蛋需要5只鸡
